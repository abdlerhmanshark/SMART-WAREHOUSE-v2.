# Smart Warehouse Management System

A professional management system for warehouse logistics developed using **C++** and featuring a modern **Web-based GUI**.

## 🚀 Overview
This project simulates real-world warehouse operations using advanced data structures. It handles:
- **Inventory Management**: Using Stacks (LIFO) for storage zones.
- **Order Processing**: Using Queues (FIFO) for customer requests.
- **Visual Dashboard**: A modern, interactive GUI to manage zones and monitor orders in real-time.

## 🛠️ Tech Stack
- **Backend Core**: C++ (Data Structures & Logic)
- **Frontend GUI**: HTML5, CSS3 (Glassmorphism), JavaScript (Tree Visualization)

## 📁 Project Structure
- `SmartWarehouse.cpp`: Core C++ implementation.
- `SmartWarehouse_GUI.html`: interactive dashboard.
- `SmartWarehouse_Pseudocode.txt`: Logic documentation.

## ⚙️ How to Run
1. Compile and run the C++ file for the backend logic.
2. Open `SmartWarehouse_GUI.html` in any modern browser to access the dashboard.
